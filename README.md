# Harcama Takip Uygulaması

Bu proje, C# WinForm ve SQLite kullanılarak geliştirilmiş bir
masaüstü harcama takip uygulamasıdır.

## Özellikler
- Kullanıcı girişi
- Gelir ve gider ekleme
- Aylık harcama takibi
- SQLite veritabanı (otomatik oluşturulur)

## Kullanım
Uygulama .exe dosyası çalıştırılarak kullanılabilir.
Veritabanı ilk çalıştırmada otomatik olarak oluşturulur.
